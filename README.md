# Project 3: Spots

## Description

Spots is a responsive profile page that displays a user's information and a gallery of photo posts. The layout adapts to different screen sizes to provide a consistent experience on mobile, tablet and desktop devices.

## Technologies Stack

- HTML
- CSS
- Responsive Design
- Figma
- Flexbox and Grid
